Создаем новый респозиторий на GitHub

Далее из консоли клонируем в нужную папку: git clone https://github.com/shihaka/название_респ.git название_папки_для_респ (папку можно не указывать). Теперь, на компьютере, в папке project  или в той, название которой указал самостоятельно, находится полная копия репозитория c GitHub. 
Чтобы начать работу с проектом надо оказаться в его директории.
